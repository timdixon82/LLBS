# LLBS logo image files

This directory holds the self-hosted LLBS logo images. Two files are needed.

## llbs-logo-with-text.png

Used by `index.html` in the header. Download from:
`https://www.llbs.co.uk/wp-content/uploads/2023/05/LLBS-logo-with-text.png`

Save as `assets/llbs-logo-with-text.png`.

## llbs-logo-brander.png

Used by `brand.html` as the branding strip logo in the canvas output. Download the source from:
`https://bd.llbs.co.uk/wp-content/uploads/2024/02/LLBS-logo-with-text-300x83-1.webp`

Convert to PNG and save as `assets/llbs-logo-brander.png`. A PNG is preferred over WEBP here
for maximum browser compatibility in the Canvas API drawImage call.

Both images are proprietary to the Lincoln and Lindsey Blind Society and are not
distributed with this repository. They must be sourced from the LLBS website.

This placeholder file is removed once the image files are committed.
